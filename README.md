# PyQt5 Image Viewer and Resizer

## Description
This project is a simple **desktop application built with PyQt5 and Pillow** that allows users to:  

- Browse and select images from their computer  
- Display the selected image in the application window  
- Resize images to a fixed dimension (300x300 pixels)  
- Save the resized image with a random filename  

This app demonstrates **basic GUI development in Python**, image handling with **Pillow**, and file dialog operations in **PyQt5**.

--

## Installation
1. Clone the repository:  
    ```bash
    git clone https://github.com/your-username/pyqt5-script-upload-resizeImage.git
    ```
2. Navigate to the project folder:
    ```bash
    cd pyqt5-script-upload-resizeImage
    ```

3. Install dependencies from the requirements.txt file:
    ```bash
    pip install -r requirements.txt
    ```
    
4. Run the application:
    ```bash
    python main.py
    ```
